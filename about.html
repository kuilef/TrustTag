TrustTag - About

TrustTag is a Chrome extension that displays warnings on websites with questionable content based on data from multiple sources.

How It Works

TrustTag automatically checks each website you visit against databases of sites with questionable content. When a match is found, a warning banner is displayed at the top of the page.

The extension periodically syncs with configured data sources to ensure you have the most up-to-date information.

Each warning shows which data source it came from, allowing you to understand the origin of the warning.

Warning Types

TrustTag can display various types of warnings, including:

Warning Type Description
False information Sites known to publish misleading or false information
Biased content Sites with heavily biased reporting or presentation
Propaganda Sites that spread political or ideological propaganda
Hate speech Sites promoting hate speech or discrimination

URL Pattern Matching

TrustTag can match websites using different patterns:

  • Exact domain match: e.g., example.com will match any page on that domain
  • Wildcard matching: e.g., wikipedia.org/*Israel* will match any Wikipedia page with "Israel" in the URL

Note: The matching is case-insensitive, so "Israel" will match "israel" or "ISRAEL" in URLs.

Data Sources

TrustTag supports multiple data sources for warnings:

  • By default, it uses our central database at https://trusttag.kuilef42.workers.dev/
  • You can add additional data sources by providing their URLs
  • Each data source can be enabled or disabled individually
  • Warnings from different sources are combined and displayed together

Each data source should return JSON data in the same format as our central database.

Settings

You can configure the following settings by clicking on the TrustTag icon in your browser toolbar:

  • Data Sources: Add, remove, or toggle data sources
  • Sync Interval: How often the extension checks for updates to the warning databases

You can also manually trigger a sync by clicking the "Sync Now" button in the popup.

Testing

You can use the included test.html file to test if your extension is working correctly.
